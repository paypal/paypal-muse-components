Welcome to our test site!

Click around to see different SDK events being triggered.

Currently this page supports "page view", "product view", "conversion", and "exclusion" events (more info on how to trigger below).

To see a page view event simply click around to different pages. You should see a page view event fired off each time you switch pages (except when you land on the home page -- this is to allow you to configure your settings before events are fired off).

To see a product view event simply click the picture associated with the product from the products pages (e.g. cats, dogs, etc).

To test OG tags, ldjson, and microdata scraping -- visit the marsupials pages. These pages are labelled with which type of meta tags we scrape.

To see a conversion or exclusion event, click the buttons on the product pages that say "purchase product" or "exclude user" respectively.

If you would like to configure where the SDK is loaded from please click the icon of the dog to the left of home. By default we load from production.
